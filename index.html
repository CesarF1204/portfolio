About Me

Professional Skills Description Ratings
HTML HTML is the skeleton of a website, it has all the basic structure and foundation on building a website. I learned HTML way back 2016 when I was a student and that time I only know how to create a link and go to another link. But now, I almost learned everything in HTML and still learning and do some practices.
CSS CSS describes how HTML elements are to be displayed on screen. Its for design and layout of HTML elements. I have known CSS since the time I learned HTML. Way back those years, I only know the basic of CSS like the background color, fonts, and opacity. Now, I have a deeply knowledge on CSS and still learning it everyday for designing my projects.
PHP PHP is a server side scripting language that is embedded in HTML. It is used to manage dynamic content, databases, session tracking, even build entire e-commerce sites. PHP is also the interaction between the website and the user. Its where you create an API or a bridge on connecting website to user especially when you encounter databases on how to create, edit, store, update, delete datas. I have also knowledge from beginner to intermediate on PHP and still learning it everyday.
Laravel Laravel is a PHP Framework and it is used in the majority of web projects, such as authentication, routing, sessions, and caching. I learned Laravel last 2019, and I like the implementation of it. Laravel is easy and better for me because it has predefined elements and its so easy to create and deploy the website. It uses MVC(Model, View, Controller) architecture pattern. You can just type and create a code to store databases and migrate it anytime you want. I also have a basic to intermediate knowledge on it. For me, Laravel is the best PHP Framework and more features are still updating.
JavaScript JavaScript is a programming language that allows you to implement complex features on web pages. It is the interaction between the website and the user. I only have a basic to intermediate knowledge on JavaScript and still learning it. I'm still confused on the OOP of JavaScript but I'm getting it on everyday practice.
ReactJS Building my first React app was an exciting experience. I learned ReactJS through the DICT training(both class and functional components). They thought as how to handle state and pass props from parent component to child components. Overall, my journey with ReactJS has been one of continuous learning and growth. I also took there MERN stack training where for my final project I built a simple app using mongodb, express, react, and nodejs.
NodeJS NodeJS is a javascript runtime environment and a backend framework that is used for server-side. I learned it from my projects that uses MySQL for database, ExpressJS, NodeJS, and ReactJS. I learned alot and discovered the flexibility of building scalable and high-performance applications using nodejs.
MySQL I've been able to explore the complexities of relational databases, develop my skills in developing and optimizing queries, and recognize the value of this reliable technology in accurately and efficiently storing and organizing data since I started studying to learn MySQL.
